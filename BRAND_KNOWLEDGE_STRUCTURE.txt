RETRIEVAL ALGORITHMS
====================

Algorithm 1: Route on Query Topic
Input: User question
Process:
  1. Extract keywords (voice, tone, design, token, naming, audit, rebrand, etc.)
  2. Match to reference_docs slugs
  3. Return matching reference doc + slug

Algorithm 2: Retrieve Matching Example
Input: User scenario description
Process:
  1. Calculate semantic similarity between user's context and example scenarios
  2. Return example with highest similarity score
  3. Extract decision framework from solution structure

Algorithm 3: Select Few-Shot Pattern
Input: User's decision type (inferred from question)
Process:
  1. Map decision type to prompt context keywords
  2. Return matching prompt
  3. Follow approach steps as conversational flow

Algorithm 4: Generate Constraint Specification
Input: Brand Agent recommendation
Process:
  1. Identify affected downstream agents
  2. Generate constraint in format: "Copy Agent MUST use [constraint]"
  3. Include validation criteria and timelines
