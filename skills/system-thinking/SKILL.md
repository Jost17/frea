# System-Thinking Skill

Systemanalyse mit Meadows (Leverage Points) und Bradley (PPV) Frameworks.

## 4-Schritt Analyse

| Schritt | Aktion | Output |
|---------|--------|--------|
| 1. System Mapping | Boundaries, Pillars, Pipelines identifizieren | System-Diagramm |
| 2. Feedback Loop Analysis | Bestehende Loops, Integritaet, fehlende Loops | Loop-Inventory |
| 3. Leverage Points | Quick Wins (L12-10), Medium (L9-7), High (L6-4), Transformational (L3-1) | Priorisierte Liste |
| 4. Optimierung | Konkrete Aenderungen, Sequenzierung, Checkpoints | Action Checklist |

## Meadows Leverage Points (Kurzform)

| Level | Typ | Beispiel |
|-------|-----|----------|
| 12-10 | Parameter/Stocks/Flows | Review-Frequenz, Inbox-Limits, Automation |
| 9-7 | Delays/Feedback | Schnelleres Feedback, bessere Constraints |
| 6-4 | Info/Rules/Structure | Dashboards, Habits, Self-Organization |
| 3-1 | Goals/Paradigms | System-Ziele, Mindset-Shift |

## Key Principles

- Systems resist change - Work with, not against
- Leverage over effort - Small changes in right place beat big changes
- Feedback is essential - Closed loops beat open loops
