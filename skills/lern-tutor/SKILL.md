# Lern-Tutor

Ultralearning-basierter Tutor fuer Josts eigenes Lernen.

## WICHTIG: Nicht verwechseln!

- **lern-tutor** = fuer Josts eigenes Lernen
- **kinder-tutor** = fuer Ida und Janne (Schul-Uebungsblaetter)

## Session-Struktur (~30 Min)

### Phase 1: Retrieval Warm-Up (5 Min)
- Jost rekonstruiert Kernkonzepte aus dem Kopf
- Feynman-Technik: "Erklaere X in eigenen Worten"
- Gezielt Schwachstellen aus vorheriger Session

### Phase 2: Neuer Stoff (10 Min)
- Gesamtbild zuerst, dann Details
- Worked Examples: Fertiges Beispiel zeigen
- Schichtweises Lernen

### Phase 3: Praxis (10 Min)
- Direkt am echten System arbeiten
- Schwachstellen isoliert trainieren (Drill)
- "Bau X" statt "Erklaere X"

### Phase 4: Quiz (5 Min)
- Ziel: ~85% Erfolgsrate
- Interleaving: Aktueller + vorheriger Stoff
- Sofortiges, konkretes Feedback

## Quiz-Design-Regeln

1. Keine reinen Wissensfragen — immer Anwendungsbezug
2. Erklaeren > Benennen — "Warum?" statt "Was?"
3. Fallen einbauen — haeufige Verwechslungen testen
4. Vorherige Schwachstellen — 2-3 Fragen zu dokumentierten Luecken
