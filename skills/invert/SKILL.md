# Invert Skill

**Charlie Mungers Inversion-Prinzip** - Probleme durch Umkehrung loesen.

> "All I want to know is where I'm going to die, so I'll never go there."

## Kernprinzip

Statt **"Wie erreiche ich Erfolg?"** → **"Was wuerde garantiert zu Misserfolg fuehren?"** → Vermeide diese Dinge.

## Der 5-Phasen-Prozess

| Phase | Aktion |
|-------|--------|
| 1. Ziel definieren | Was willst du erreichen? Erfolgs-Kriterien? |
| 2. Invertieren | "Wenn ich WOLLTE, dass es scheitert - was wuerde ich tun?" |
| 3. Kategorisieren | Sofort-Killer / Schleichender Tod / Vermeidbare Fehler / Externe Risiken |
| 4. Schutz ableiten | Fuer jeden Killer: konkrete Praeventionsmassnahme |
| 5. Synthese | Zurueck zum Positiven mit neuen Insights |

## Quick-Start

```markdown
## Inversion: [Thema]
### Ziel: [Was will ich erreichen?]
### Was wuerde GARANTIERT scheitern?
1-5 Killer auflisten
### Also muss ich:
1-5 Gegenmassnahmen
```
