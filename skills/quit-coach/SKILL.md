# Quit Coach

**Annie Duke's Framework** - Strategisches Aufgeben als Kompetenz.

> "Quitting is not the opposite of success. Quitting is a tool that helps you find success."

## Kernprinzip

**Grit wird ueberschaetzt.** Die meisten halten zu lange fest (Sunk Cost), verwechseln Aufgeben mit Versagen, ignorieren Opportunity Costs.

## Der Quit-Check (6 Phasen)

| Phase | Frage |
|-------|-------|
| 1. Kontext | Was steht zur Debatte? Wie lange schon? |
| 2. Erwartungen | Was war das Ziel? Erreicht? |
| 3. Sunk Cost | Was haelt dich - Fakten oder investierte Zeit/Geld? |
| 4. Opportunity Cost | Was kostet dich das Weitermachen? |
| 5. Killer-Frage | Wuerdest du HEUTE nochmal anfangen? |
| 6. Empfehlung | WEITERMACHEN / AUFHOEREN / KURSKORREKTUR |

## Quick-Start: Die 5 Fragen

1. Wie lange machst du das schon?
2. Urspruengliches Ziel erreicht?
3. Was haelt dich - Fakten oder Sunk Cost?
4. Was kostet Weitermachen?
5. Wuerdest du HEUTE nochmal anfangen?

**Auswertung:** 2+ "Nein/Unsicher" → Signal fuer Aufhoeren
